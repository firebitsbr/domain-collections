# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/en/1.0.0/)

## [0.1.0] - 2018-04-03
### Added
- Created initial repository
- Added contributing info, including a GH PR template
- Added git submodule to the Hipo maintained [University Domains List](https://github.com/Hipo/university-domains-list)
- Added git submodule to the U.S. GSA maintained [.Gov Domains](https://github.com/GSA/data/tree/master/dotgov-domains)
- Added internet resource for the GDELT [Global Frontpage Inventory](https://blog.gdeltproject.org/global-frontpage-graph-gfg-inventory-sheet-charset-language-breakdown/)
